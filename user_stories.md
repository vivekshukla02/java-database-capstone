# Smart Clinic Management System - User Stories

### 1. Doctor User Stories
* **As a Doctor**, I want to view my daily schedule and appointment list so that I can manage my time and prepare for patients efficiently.
* **As a Doctor**, I want to update my available time slots dynamically so that patients only book slots when I am available.
* **As a Doctor**, I want to create and save digital prescriptions for patients during a consultation to maintain accurate digital health records.

### 2. Patient User Stories
* **As a Patient**, I want to search for doctors by their name or medical specialty so that I can find the right healthcare provider for my needs.
* **As a Patient**, I want to view available time slots for a specific doctor and book an appointment online easily.
* **As a Patient**, I want to log into my personal portal to view all my past appointments and medical prescriptions securely.

### 3. Admin User Stories
* **As an Admin**, I want to add, update, or remove doctor profiles (including credentials and specialty) to keep the system database accurate.
* **As an Admin**, I want to generate daily appointment reports across the clinic to monitor operational efficiency and patient flow.
